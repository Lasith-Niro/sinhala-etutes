Reason to stop working of camera

In my point of view this problem encountered with Android OS. But in early android versions (Example: 2.3) this problem never occurred. But in KitKat or newly released OS this problem occurs.
I found some methods to avoid this issue temporally. Examples:

  • clearing camera app cache
  • uninstall camera updates
  • disable/re-enable app
  • restarting the phone
But in normal usage users can't ignore the issue even they update their OS.So I think this is an OS level problem and It needs an OS level answer. So users can use software to change permissions in apps. One such solution is users can use "Permission Explorer".
