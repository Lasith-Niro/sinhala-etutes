Project Details

Project Title : Searching blocks in the Blocks editor
Brief: When working on bigger projects, it can be rather difficult to find blocks. A way of searching and focusing on certain blocks would be very useful.

The Goal and Objectives

Implementing a way of searching and focusing on specified blocks.
  1. Improve the prototyping work that has been already carried out.
  2. Provide search list relevant to the entered block.

Problem Definition

When dealing with complex projects having a large number of blocks, it is very difficult to handle them due to the reason that there is no specific method to search blocks. It will be easier for the user to focus on required blocks if there is a way to search them.

Brief introduction to the project

A prototype search box to a find a required blocks has already been implemented up to now. The main objective is to implement the functionality of the prototype and enhance the usability.

Requirements

Functional
  1. Providing a search box for the user to enter the required blocks
  2. Visualizing a search list to display the relevant search results.
Non-Functional
  1. Should be easy to use.
  2. Should be using best web standards

Main deliverables of the system

  1. Source codes of the module
  2. Test cases and test data
  3. Documentations

3rd Party tools and libraries

  1. Blockly
